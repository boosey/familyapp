If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Kindred specifics:
- Elders-first: 18px UI floor, 22px story floor, 64px default touch target, 96px voice action.
- Voice over keyboard: the mic is the one loud control per screen; typing is always offered, never required.
- Newsreader (serif) for stories, Public Sans (sans) for interface, mono only for years/metadata.
- Theme via data-theme: heirloom (default), archive, hearth.
- Warm paper backgrounds + single accent; warm brown shadows, never gray. Stories are the hero; chrome recedes.
