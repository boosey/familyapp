Provenance tags for people, places and years. Use in rows under a story or in a tag picker.

\`\`\`jsx
<KindredChip kind="person" label="Maria" avatar="sage" />
<KindredChip kind="person" label="Tony" avatar="accent" />
<KindredChip kind="place" label="Amalfi" />
<KindredChip kind="time" label="1962" />
\`\`\`

`person` shows an avatar initial; `place` prefixes a pin; `time` renders in mono.
