Family Chronicle · Capture design · for review

Three capture flows, one composing surface

Every place a person puts words into the system — filling in intake, telling a story unprompted, or answering a question someone asked — shares the same working surface: record or type → (transcribe → cleanup, for voice) → edit, with recording and typing that stay live so you can keep going, freely mixed. What differs is only the tail: intake extracts a fact; a story gets finished, consented, and shared.

Mark up anything — states, transitions, or what gets written. Nothing here is built yet.

Phases (state colour)

DRAFT / composing— the live working surface: record, append, edit, polish
PENDING_APPROVAL— finished; confirm title + audience, then consent
SHARED— consented and visible to the family
EXTRACTED— intake terminal: a fact saved, never shared

Operations & rules (shared)

two entry types — a voice take (audio → Transcription → Cleanup) or a typed take (words are the content, user_authored; skips Transcription+Cleanup). Freely interleaved; any audio ⇒ voice-kind.
automatic, per take — Cleanup fixes filler, false starts, within-take self-corrections (keeps the hedge if unsure). Sees one take only.
human-confirmed, holistic — ✨ Polish (de-rambles + resolves cross-take self-corrections) via the button or the Finish check; never silent, always reversible, logged. hand-edit is yours & permanent.
audio vs. truth — audio = permanent original record (never discarded; playback/audit/improvement). Source of truth = the approved prose (composite; authored, not regenerable from audio).
retention & ledger — all audio kept (intake too); only Discard/Drop delete; append never re-touches earlier text. Every stage appends to the prose lineage.
memory extraction — every mode mines what was said into system memory (anchors now, broader narrator memory deferred). Retention is unconditional; memory is consent-gated — a Story feeds memory only post-approval; intake extracts at Save (answering = consent).
Intake

Filling in a biographical answer

One structured question at a time (“Where did you grow up?”). Same composing surface — but it is not a Story: no interviewer follow-ups, no audience, no consent, never surfaced. Audio & transcript are kept for audit and prompt improvement — and intake is the designated first source of narrator memory (seam ready via retention; the memory model itself is deferred).
